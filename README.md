# Getting Started

## Step 1

1. Install Composer
2. Install Node

## Step 2

1. **Clone the repository:**

    ```bash
    git clone https://github.com/lequanphat/project_name
    ```

2. **Change to the project directory:**

    ```bash
    cd project_name
    ```

3. **Install Composer dependencies:**

    ```bash
    composer install
    ```

4. **Install Node.js dependencies:**

    ```bash
    npm install
    ```

## Step 3

Run the application:

```bash
php artisan serve
```

or

```bash
run
```
